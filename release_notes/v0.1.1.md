# Release Notes — v0.1.1

AI Ops SOP Pack v0.1.1 is a documentation visibility and status alignment release.

It aligns the public repository documentation after the v0.1.0 release and clarifies the main external use cases.

## Release type

Documentation-only release.

## What changed

- README now includes a `Use this when` section.
- README now highlights `content/04_agent_handoff_format.md` as the best first file to read.
- START_HERE is aligned with the V0.1 public documentation release.
- START_HERE now repeats the positive use-case framing.
- Current repository status wording reflects the public V0.1 documentation release state.

## What did not change

v0.1.1 does not add:

- runtime capability
- provider integration
- live execution
- background daemon / watcher
- memory or ledger writes
- trading, order routing, leverage, wallet, or portfolio authority
- product offer
- commercial channel
- PDF distribution
- external announcement
- permission-to-act

## Recommended use

Start with:

1. `README.md`
2. `START_HERE.md`
3. `content/04_agent_handoff_format.md`
4. matching template from `templates/`
5. `examples/pr_crash_recovery_filled_example.md` if a filled walkthrough is needed
